Github használatának legfontosabb parancsai

  • git add: Fájlok előkészítése (staging)
  • git commit: Változtatások megnevezése verziókövetéshez
  • git push: Helyi változtatások feltöltése
  • git pull: Külső változtatások letöltése
  • git clone: Repository helyi mentése
  • git branch: Új branch létrehozása
  • git checkout: Branch-re váltás
  • git status: Repository állapota

Local / Remote / VS Code felhasznált bővítmények

  • Live Server: HTML dokumentum megtekintése localhost-ban
  • Prettier: Kód formázás
  • ESLint: Részletes hibaüzenetek debugoláshoz
  • GitLens: Részletesebb Git interfész VS Code-on belül
